*Tribute Page for Mother Teresa:::
A simple and responsive tribute page created using HTML and CSS, dedicated to Mother Teresa, a humanitarian known for her compassion and selfless service to humanity.
*Features::
Clean and elegant design
Fully responsive layout for both desktop and mobile
Brief biography of Mother Teresa
Inspirational quotes and achievements section
Beginner-friendly and well-structured HTML/CSS code
*Technologies Used
HTML5
CSS3
*Inspiration
This project was created to improve web development skills while paying tribute to Mother Teresa, whose life and work continue to inspire millions around the world.
*How to Run Locally
1. Clone this repository:
git clone https://github.com/mamtashah487/tribute-page.git
2. Open the index.html file in your web browser.
